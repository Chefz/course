Тестирование

1. С помощью чего осуществляется создание базы данных

DDL (Data Definition Language) запросы. DML (Data Manipulation Language) запросы.

2. Как осуществить подключение базы данных к приложению

Используя JDBC (Java Database Connectivity). Используя HTTP-запросы.

3. Какие СУБД применяются для разработки

MySQL, PostgreSQL, Oracle. JavaScript, HTML, CSS.

4. Какие отношения между сущностями существуют

Один к одному, один ко многим, многие ко многим. Синтаксические, семантические, структурные.

Практическое задание

На основе созданного приложения на Spring Boot создайте базу данных с не менее чем тремя таблицами и связями между ними.

Используйте созданное хранилище данных для хранения информации в базе данных.
